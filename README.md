* **01 Stochastic Episode Sampler:** This script acts as a weighted randomizer for TV show episodes. Instead of a uniform probability distribution, it assigns selection weights based on IMDb ratings.
  Mathematical Model:
> Ratings $$\ge$$ 7.2 operate in a "ground state" with a uniform probability weight of 1.0 Whereas, Ratings < 7.2 operate in a high-energy state, where the probability weight decays exponentially as the rating drops, simulating a Boltzmann-like distribution.
